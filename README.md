# Sudoku Solver using Backtracking (Java)

This project implements a Sudoku Solver using the Backtracking algorithm in Java.

## Features
- Solves a standard 9x9 Sudoku puzzle
- Uses recursive backtracking
- Validates row, column, and 3x3 subgrid constraints

## Algorithm
The solver tries numbers from 1 to 9 in empty cells and checks if placing a number is valid. If a conflict occurs later, it backtracks and tries another number.

## Concepts Used
- Recursion
- Backtracking
- 2D Arrays
- Constraint Checking
